## 🛠️ Technology Stack
This project leverages the latest industry-standard tools for speed, scalability, and performance:

- **Build Tool**: [Vite](https://vitejs.dev/)

## 📁 Repository Structure
To maintain a professional environment, the code is organized for maximum readability and scalability:

```bash
├── public/              # Static assets (Images, Resume, Icons)
├── src/
│   ├── components/      # Atomic UI components (Buttons, Cards, Inputs)
│   ├── sections/        # Main page sections (Hero, About, Experience)
│   ├── lib/             # Utility functions and shared libraries
│   ├── App.tsx          # Main application entry and routing
│   └── index.css        # Global styles and design system tokens
├── tailwind.config.js   # Custom design tokens (colors, fonts, animations)
└── vite.config.ts       # Optimized build configuration
```
